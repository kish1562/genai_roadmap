# 0 → GenAI Engineer

Four phases, ~24 weeks. Build order, not a reading list — each phase only unlocks what the next one needs.

Edit this file directly to track progress: change `- [ ]` to `- [x]` for anything you finish, then commit + push. GitHub renders these as real checkboxes.

---

## Phase 1 — Python + SQL Foundations (Weeks 1–4)

**Week 1**
- [x] Install Python + set up VS Code
- [x] Variables, data types, control flow (if/else, loops)
- [x] Write 5 tiny scripts (temp converter, FizzBuzz, etc.)

**Week 2**
- [x] Functions, lists, dicts, string methods
- [ ] Read/write files (txt, csv)
- [ ] Error handling (try/except)

**Week 3**
- [ ] pip + installing packages, basic OOP (classes)
- [ ] requests library — call a public API and print results
- [ ] Start SQL: SELECT, WHERE, ORDER BY

**Week 4**
- [ ] SQL: JOINs, GROUP BY, subqueries, window functions basics
- [ ] Combine: pull data via API, save to file, query with SQL
- [ ] Checkpoint: comfortable writing a script from scratch, no tutorial

> Skip: Django/Flask deep-dives, DSA/LeetCode grinding, DBA internals (indexing, replication).

## Phase 2 — AI Fundamentals (Weeks 5–9)

**Week 5**
- [ ] What LLMs are (high level, no deep math)
- [ ] DeepLearning.AI short course: Prompt Engineering
- [ ] Practice: 10 prompts solving real mini-tasks

**Week 6**
- [ ] OpenAI / Azure OpenAI API — first API call from Python
- [ ] Understand tokens, temperature, system prompts
- [ ] Build a tiny CLI chatbot (no framework)

**Week 7**
- [ ] Embeddings — what they are, why they matter
- [ ] Vector DB basics — pick one (Chroma) and store/query vectors
- [ ] Build: embed 10 sentences, find nearest neighbors

**Week 8**
- [ ] RAG — the concept end to end
- [ ] LangChain basics: chains, retrievers, prompts
- [ ] Wire together: load a doc → embed → retrieve → answer

**Week 9**
- [ ] Review + fill gaps
- [ ] Checkpoint: can explain RAG out loud in 60 seconds, no notes

> Skip: neural nets from scratch, calculus/linear algebra courses, PyTorch/TensorFlow, Kaggle.
